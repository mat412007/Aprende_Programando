¿Cómo se escribe CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación y el diseño de un documento HTML o XML. Se utiliza para definir cómo se ven los elementos y componentes de una página web, incluyendo el color, la tipografía, el tamaño, el espaciado, el diseño y otros aspectos visuales. El primer paso en este curso fue completar el esqueleto del sitio web. Ahora, llega el momento de darle color a todo esto. El primer paso es conocer la estructura de CSS, cómo se escribe. Luego, veremos cómo incorporarlo al esqueleto.
Cada característica tiene un selector, propiedad, y valor.
h1{
color: red;
}
Por ejemplo:

  • text-decoration: underline;
  • color: grey;
  • border: 2px solid red;
  • background-color: goldenrod;

Implementando CSS

Existen varias maneras de relacionar, entrelazar, juntar, linkear el esqueleto del sitio con su cuerpo, el HTML con el CSS, y varias de estas maneras son incorrectas. Aun así es importante conocerlas ya que puede suceder que te topes con algún sitio web que aún las utilice o que al momento de trabajar sobre otro proyecto ya existente tenga estas metodologías. En este apartado veremos las formas correctas e incorrectas de implementar CSS.
Hay 3 maneras de aplicar CSS:

  • Estilos en línea
  • La etiqueta <style>
  • La etiqueta <link>
La etiqueta <link> tiene 2 atributos:
  • rel:"": especifíla relación entre el documento actual, y el recurso entrelazado.
  • href:"": especifíca la ubicación del recurso externo que se va a enlazar.

Propiedades para textos y fondos

En este tercer tema veremos con un poco más de profundidad y detenimiento algunas propiedades más utilizadas que son las de color y fondo. Ambas son las primeras en retocarse y las que comienzan a definir un poco la “personalidad” o el “humor” del sitio web. Una paleta de colores es una colección de colores seleccionados y organizados de manera armoniosa. Se utiliza en diseño gráfico, diseño web y otras disciplinas creativas para elegir y combinar colores de forma coherente y estética. Además, ayuda a transmitir emociones, establecer la identidad de una marca y crear una experiencia visual cohesiva. Teniendo una paleta seleccionada es más fácil su aplicación desde el código. Por otro lado también se verá cómo modificar otros aspectos de los fondos y otros aspectos de los textos, como su tamaño, su peso, etc.
Las propiedades de texto son:

  • font-family: tipo de letra.
  • font-size: tamaño de letra.
  • font-weight: grosor de la letra.
  • font-style: estilo del texto, como cursiva o inclinada.
  • text-align: alineación del texto, izquierda, derecha, centrado, o justificado.
  • text-decoration: agrega decoraciones al texto, como subrayado o tahcado.
  • text-transform: cambia la capitalización del texto.
  • line-height: altura de las líneas de texto.
  • letter-spacing: espacio entre los carácteres del texto.
Las propiedades de fondo son:
  • background-color:
  • background-image: url()
  • background-repeat:
    • repeat; la imagen se repite horizontal y verticalmente.
    • repeat-x; la imagen se repite horizontalmente.
    • repeat-y; la imagen se repite verticalmente.
    • no-repeat; la imagen no se repite.
  • background-position: especifíca la posición inicial de la imagen de fondo, ya sea; left, right, top, bottom, center.
  • background-size: especifica el tamaño de la imagen de fondo, ya sea en pixeles, porcentaje, auto, cover, contain.
  • background-attachment: indica si la imagen de fondo se desplaza con el contenido de la página o no.
    • scroll; la imagen se desplaza con el contenido.
    • fixed; la imagen se mantiene fija en su posición.
    • local; la imagen de fondo se desplaza junto con el contenido de su contenedor.

Clases

Ya pusimos en práctica algunas propiedades y como vimos, muchos elementos cambian características a la vez. Quizás existan partes en específico que no queramos que se vean así, como todos los <h2>, o todo los <p>. Para hacer que no todos estos elementos tengan las mismas características se utilizan las clases, que son selectores especiales asignados a cada elemento en particular para que posea propiedades únicas. Son muy fáciles de usar y serán necesarias a lo largo de todo el curso.
