Posicionamientos

Ahora vamos a ver la propiedad de CSS: position.
Esta propiedad se utiliza para ver de como se posiciona un elemento en comparación de los otros. Sus valores posibles son:

  • static: los elementos con este valor se posicionan según el flujo normal del documento, y propiedades como left, right, top y bottom no los afectan.

  • relative: los elementos se posicionan según su posición normal. Las propiedades como left, right, top, bottom tienen efecto según su posición original.

  • absolute: los elementos con esta posición se posicionan en relación con el elemento padre que tenga una posición distinta a static. Si no existe se toma el elemento body. De esta manera se puede definir la posición exacta del elemento mediante el uso de left, roght, top, bottom.
    Para centrar por completo un elemento en este tipo de posición, se usa la propiedad calc() de CSS y decimos el objeto se mueva un 50% menos la mitad del elemento. Por ejemplo: left: calc(50% - 50px).

  • fixed: los elementos con esta posición se quedan fijos en la página incluso si esta se mueve, todo en relación de la ventana del navegador. Para setear su posición se usa left, right, top, y bottom.

  • sticky: los elementos con esta posición se comportan como relative hasta que en cierta posición de la ventana del navegador se vuelven fixed. Esta posición es seteada con top, bottom, right y left.

Tambien tenemos la propiedad z-index. Esta controla la posición de los elementos en el eje z, determina como y en qué orden se apilan. Su valor puede ser un número entero o la palabra clave auto.
